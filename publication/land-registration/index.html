Land Registration System Using Blockchain - A Decentralized Application with IPFS & Ethereum

Abstract

Blockchain is a distributed ledger system that maintains track of each of the transactions that have taken place on a peer-to-peer network. The blockchain serves as an immutable ledger that allows transactions to happen using a decentralized approach. When you buy land, getting it registered is a top priority as it provides you legitimacy to carry out any transaction. A person is only considered the legal owner of a property after he gets the property registered in his name.

In India, many reforms have been introduced within the land registration system to make it better. Still, the current system has a lot of vulnerabilities, which also causes delays. This paper investigates and suggests a distributed system using blockchain that can keep track of all the transactions completed throughout the process of buying land and can be a lot faster and more secure because of the fewer processes involved. The proposed architecture also makes it feasible to validate the land using immutable transactions that are kept on the public ledger.
